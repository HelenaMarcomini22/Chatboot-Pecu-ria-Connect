🚀 Como Executar

bash
=> Clone o repositório
git clone https://github.com/HelenaMarcomini22/chatbot-pecuaria.git
cd chatbot-pecuaria

=> Instale as dependências
pip install -r requirements.txt

=> Configure sua chave da API Gemini
export GOOGLE_API_KEY="sua-chave-aqui"

=> Execute o projeto
python main.py
